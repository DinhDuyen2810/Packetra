Packetra Quick Start

This page is a short quick-start guide. It assumes you already downloaded the source code and only need the basic steps: enter the project folder, install libraries, run the app, and package it.

Source already downloaded Install packages Run Package

1. Open the Project Folder

cd DATN-Packetra

If you use a virtual environment, activate it before installing packages.

2. Install Libraries

pip install -r requirements.txt

This installs the GUI, packet analysis, and AI runtime dependencies used by the project.

3. Run the App

python main.py

If the main window opens, the project is ready to use.

4. Package the App

One simple Windows packaging option is PyInstaller.

pip install pyinstaller
pyinstaller --noconfirm --clean --windowed --name Packetra ^
  --add-data "image/ava.png;image" ^
  --add-data "ai;ai" ^
  --add-data "data;data" ^
  --add-data "demo;demo" ^
  --add-data "help;help" ^
  --add-data "image;image" ^
  --collect-all PySide6 ^
  main.py

This command explicitly bundles image/ava.png together with the rest of the image folder, so the application avatar is available in the packaged build.

After build completes, check the packaged app in dist/Packetra/.

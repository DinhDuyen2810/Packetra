3. Step-by-Step Display-Filter Entry Paths

Main filter bar

  1. Locate the filter bar above the packet list.
  2. Type an expression.
  3. Press Enter or click the arrow button.
  4. Use the clear button to reset it.

History and autocomplete

  1. Click the trailing drop-down icon inside the filter input to open recent filters.
  2. Choose an older expression to reapply it immediately.
  3. While typing, use autocomplete suggestions for protocols, fields, and operators.

Drag and drop from packet list

  • Source -> creates src == "value"
  • Destination -> creates dst == "value"
  • Protocol -> creates protocol == "value"
  • Length -> creates frame.len == number

Packet list right-click context menu

  1. Right-click a packet-list cell.
  2. Open Apply as a Filter.
  3. Choose one of the merge modes such as Selected, Not Selected, AND, OR, AND NOT, or OR NOT.

Packet details right-click or Analyze menu

  1. Select a field in the packet details tree.
  2. Either right-click and use Apply as a Filter, or open Analyze > Apply as Filter.
  3. Packetra builds a detail-based filter from the selected node.

Expression builder

  1. Open Analyze > Display Filter Expression....
  2. Choose a field.
  3. Choose a relation.
  4. Enter a value.
  5. Choose whether to replace, AND, OR, AND NOT, or OR NOT with the current filter.

Macros, conversation filters, and Follow Stream

  1. Use Analyze > Display Filter Macros... for reusable named expressions.
  2. Use Analyze > Conversation Filter to build scoped expressions for TCP, UDP, IPv4, IPv6, or Ethernet conversations.
  3. In Follow views, use Filter This Stream or Filter Out This Stream.
